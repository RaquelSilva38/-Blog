Algoritmo e lógica de Programação

Lógica de Programação é a técnica de desenvolver algoritmos (sequências lógicas) para atingir determinados objetivos dentro de certas regras baseadas na Lógica matemática e em outras teorias básicas da Ciência da Computação e que depois são adaptados para a Linguagem de Programação utilizada pelo programador para construir seu software.

Exercício if

If é uma expressão condicinal cujo valor poder ser falso ou verdadeiro. Na linguagem C falso é representado por 0 e verdadeiro é representado por 1. No exercício abaixo, o usuário dgita sua idade, a condição If verifica se o usuário é maior de idade e a else se é menor.

Exercício While

Executa a repetição de um bloco de instruções enquanto uma condição é verdadeira.
